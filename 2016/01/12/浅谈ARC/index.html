1. 引用计数

引用计数可以有效管理对象生命周期，当alloc一个对象时，其引用计数为1；当有其他对象指向该对象时，
引用计数+1；当指向对象release时，引用计数-1，直至引用计数为0时，我们就将该对象销毁，回收内存。

我们可以通过下面的列子来看引用计数的变化：

`

int main(int argc, const char * argv[]) {

NSObject *object = [[NSObject alloc]init];
NSLog(@"Reference Count = %lu", (unsigned long)[object retainCount]);

NSObject *another = [object retain];
NSLog(@"Reference Count = %lu", (unsigned long)[object retainCount]);

[another release];
NSLog(@"Reference Count = %lu", (unsigned long)[object retainCount]);

[object release];
return 0;
}

`
